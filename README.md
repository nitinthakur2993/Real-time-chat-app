# 💬 Real-Time Chat Application

A modern, full-stack real-time messaging application that allows users to join custom rooms and chat instantly with friends. Built with security, speed, and scalability in mind.

---

## 🚀 Features

*   **Real-Time Messaging:** Instant message delivery using WebSocket protocols.
*   **Custom Rooms:** Users can create or join specific rooms to isolate their conversations.
*   **Secure Authentication:** Designed to support Gmail and Phone OTP secure logins (under active development).
*   **Persistent Chat History:** Integrated with MongoDB/Supabase to store and load messages permanently.
*   **Sleek UI/UX:** Responsive interface built for both desktop and mobile viewing.

---

## 🛠️ Tech Stack

**Frontend:**
*   React.js (Vite)
*   HTML5 & Custom CSS3
*   Socket.io Client

**Backend:**
*   Node.js & Express
*   Socket.io (WebSockets)
*   Mongoose / MongoDB (Database)
